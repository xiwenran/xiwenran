## 👋 关于我

你好,我是 **xiwenran(西里)**,一名**内容型业务创业者** + **独立开发者**。

主业做**教师虚拟资料**这条赛道,平台是小红书 + 抖音,私域承接转化,已经跑了一年多 —— 把"一个人能不能跑通闭环"这件事验证完了,现在在做的是**把所有重复劳动变成 AI 工具**,让单人团队跑出小团队的产能。

日常重度使用 **Claude Code**,自己搭了一套**三模型分工**(Opus 4.7 决策 + Codex GPT-5.4 写代码 + Sonnet 4.6 写文档)和**多 Agent 基础设施**(冷眼审查 / 圆桌讨论 / Obsidian 自动捕获),所有规则都物理化落到 `CLAUDE.md`,不靠记忆靠机制。

> 相信的事 —— **工具的复利 > 短期爆量**,**把打法做成产品 > 单纯执行**,**一层一层来** — 卖资料 → 卖效率 → 卖方法 → 卖信任。
